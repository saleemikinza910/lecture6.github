🎮 ChronoAge · Age + Arcade Studio

An interactive web application that combines an Age Calculator, Utility Functions, and Mini Games in a single project.

Built using HTML, CSS, and Vanilla JavaScript.

---

 📌 Project Overview

ChronoAge is an all-in-one web app where users can:

- Enter personal details (name, birth year, favorite movie)
- Calculate their age instantly
- Check voting eligibility
- Explore basic programming concepts
- Play mini games (strategy + casual)

---
 ✨ Features

🧾 Age & Identity System
- User inputs:
  - Name
  - Birth Year
  - Favorite Movie
- Outputs:
  - Calculated Age
  - Dynamic Introduction
  - Voting Eligibility Status
  - Next Milestone (next decade)
  - Generation Category

---
 ⚙️ Utility Functions

➕ Add Two Numbers  
Simple function to calculate sum of two numbers.

🔄 For Loop (1 → 10)  
Displays numbers from 1 to 10 using loop.

⚖️ Positive / Negative Check  
Classifies number using if/else:
- Positive
- Negative
- Zero

---
 🎲 Game Zone

 🧩 Mastermind (Strategy Game)
- Guess a 4-digit secret code (no repeats)
- 🔥 = correct position
- 💧 = correct digit wrong position
- 10 attempts limit

🎨 Color Flipper (Casual Game)
- Click button to generate random color
- Changes background dynamically

---

🌙 Theme System

- Dark / Light mode toggle
- Saved using LocalStorage
